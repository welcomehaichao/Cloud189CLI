# 天翼云盘 CLI - 快速参考

## 安装

```bash
# 编译
go build -o cloud189.exe ./cmd/cloud189

# 或下载预编译版本
# Windows: cloud189.exe
# Linux/Mac: cloud189
```

## 登录命令

### 密码登录
```bash
cloud189 login -u 用户名 -p 密码
```

### 二维码登录（推荐）
```bash
cloud189 login --qr
```

### 其他
```bash
cloud189 whoami    # 查看当前用户
cloud189 logout    # 退出登录
```

## 文件操作

### 列出文件
```bash
cloud189 ls /              # 根目录
cloud189 ls /文档          # 绝对路径
cloud189 ls 文档           # 相对路径
cloud189 ls /文档 -l       # 详细列表
cloud189 ls / -o table    # 表格格式
cloud189 ls / --family    # 家庭云
```

### 创建文件夹
```bash
cloud189 mkdir 新文件夹             # 根目录创建
cloud189 mkdir /文档/新文件夹       # 指定路径创建
cloud189 mkdir /文档 --family      # 家庭云创建
```

### 查看容量
```bash
cloud189 info
cloud189 info -o table    # 表格格式
```

### 创建文件夹
```bash
cloud189 mkdir 新文件夹
cloud189 mkdir /文档/子文件夹 --family
```

### 查看容量
```bash
cloud189 info
```

## 家庭云

```bash
cloud189 family list      # 列出家庭云
cloud189 ls / --family    # 家庭云文件列表
```

## 输出格式

```bash
-o json    # JSON格式（默认）
-o yaml    # YAML格式
-o table   # 表格格式
```

## 配置文件

位置: `~/.cloud189/config.json`

```json
{
  "username": "用户名",
  "session_key": "...",
  "refresh_token": "..."
}
```

## 常见问题

### 二维码无法识别？
1. 确保使用 v1.0.1+ 版本
2. 终端使用等宽字体
3. 调整手机与屏幕距离（10-20cm）
4. 或使用浏览器打开链接

### 登录失败？
1. 检查网络连接
2. 尝试二维码登录
3. 删除配置文件重新登录

## 示例工作流

### 首次使用
```bash
# 1. 登录
cloud189 login --qr

# 2. 查看根目录
cloud189 ls /

# 3. 创建文件夹
cloud189 mkdir 备份

# 4. 查看容量
cloud189 info
```

### 日常使用
```bash
# 查看文件
cloud189 ls /文档

# 创建新文件夹
cloud189 mkdir /文档/新项目

# 查看登录状态
cloud189 whoami
```
